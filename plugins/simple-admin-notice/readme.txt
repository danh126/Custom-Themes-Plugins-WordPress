== Description ==
Hiển thị thông báo đơn giản trên trang admin và cung cấp shortcode cho bài viết.

== Installation ==
1. Tải file ZIP và cài đặt qua trang quản trị WordPress.
2. Kích hoạt plugin.
3. Sử dụng shortcode `[san_shortcode message="Hello!"]` trong bài viết.
